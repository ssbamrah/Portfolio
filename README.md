## 🧪 Featured Project

### Secure Multi-VLAN Enterprise Infrastructure & Vulnerability Assessment Lab

> Designed and deployed a routed enterprise-style virtualized infrastructure across multiple VLANs using Ubuntu Server and Windows Server Active Directory.

**Key Implementations**

- Segmented multi-VLAN architecture with inter-VLAN routing
- NAT and firewall policy configuration
- Active Directory Domain Services deployment
- DNS and DHCP server configuration with reservations
- Hosted WebGoat and Magento web applications
- Vulnerability scanning via OWASP ZAP and Nessus Essentials
- Packet analysis and findings validation with Wireshark
- Restricted access controls for vulnerable/test systems

**Technologies Used**

`Ubuntu Server` `Windows Server` `Active Directory` `Wireshark` `OWASP ZAP` `Nessus` `VLANs` `DNS` `DHCP` `Firewalling`
